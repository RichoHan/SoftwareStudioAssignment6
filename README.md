## I. Explanation of the Design

### Operation
+ 點擊按鈕"Add All Nodes"，使用者可以將當前的所有Characters都加入Network。
+ 點擊按鈕"Clear All Nodes"，使用者可以將當前的所有Characters都移出Network。
+ 當把滑鼠移到任一Character時，Character會變大、且顯示出其名稱資訊。
+ 可以使用滑鼠按住來拖移Character，可藉此移入或移出Network。
+ 當按下鍵盤的數字1~7，可以切換到對應的集數。

### Visualization
+ 當有多個Characters在Network上時，彼此之間會顯示有連線，其連線的寬度會根據連線的值越大而越粗。
+ 點擊按鈕"Clear All Nodes"，會顯示Characters移出Network的動畫。
+ 當按下鍵盤的數字1~7、成功拖移Character或是點擊按鈕時，會有音效輸出。
